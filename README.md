# Monochrome Player

Minimalist music player with audio visualizer.

## Versions

### Desktop (Tauri + Rust)
Standalone application for Windows.

**Branch:** [`Desktop`](../../tree/Desktop)

**Features:**
- F11 fullscreen
- Custom backgrounds
- Scan local folders
- No browser required

**[Download Latest Release](../../releases/latest)**

---

### Web (React)
Browser-based version.

**Branch:** [`web`](../../tree/web)

**Features:**
- Works in any modern browser
- Drag & drop files
- Three color themes
- Audio visualizer
