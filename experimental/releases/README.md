# Monochrome Player - Releases

## Portable Version

This is a portable music player. No installation required.

### Requirements
- Windows 10/11 with WebView2 Runtime (comes pre-installed on Windows 11 and recent Windows 10 updates)

### Usage
1. Download `MonochromePlayer-portable.zip`
2. Extract to any folder
3. Run `Monochrome Player.exe`

### If WebView2 is not installed
Download from: https://developer.microsoft.com/en-us/microsoft-edge/webview2/
