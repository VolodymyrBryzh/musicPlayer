Monochrome Player v0.1.3

PORTABLE: Extract ZIP, put music in this folder, run exe.

CONTROLS:
- F11 / Double-click background: Fullscreen
- Escape: Exit fullscreen
- Scan All: Find music in this folder
- Select Folder: Choose other folder

SETUP:
1. Extract all files to one folder
2. Put music (MP3, FLAC, WAV) here or in subfolders
3. Put backgrounds (PNG, JPG) in /backgrounds
4. Run monochrome-player.exe
