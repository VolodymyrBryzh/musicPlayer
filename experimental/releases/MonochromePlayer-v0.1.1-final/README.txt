Monochrome Player v0.1.1

PORTABLE: Extract, add music, run.

CONTROLS:
- F11: Fullscreen toggle
- Escape: Exit fullscreen
- Double-click background: Fullscreen
- Scan All: Find music in this folder
- Select Folder: Choose folder

SETUP:
1. Extract all files
2. Add music (MP3, FLAC, WAV) to this folder
3. Add backgrounds (PNG, JPG) to /backgrounds
4. Run monochrome-player.exe
